Overview

The base-class for all Ronin Web Servers. Extends Sinatra::Base with additional routing methods, helper methods and Sinatra conditions.

Routing Methods

  • any: registers a route that responds to GET, POST, PUT, PATCH, DELETE and OPTIONS requests.
  • default: registers the default route.
  • file: mounts a file at the given path. a given file.
  • files: monts multiple files. given paths.
  • directory: mounts a directory at the given path.
  • directories: mounts multiple directories.
  • public_dir: mounts a directory at the root.
  • host: mounts a Rack app for the given vhost.
  • map: mounts a Rack app at the given path.
  • proxy: mounts a Proxy at an optional path.

Helper Methods

  • h: escapes HTML entities.
  • file: sends a file.
  • mime_type_for: returns the MIME type for the file.
  • content_type_for: sets the Content-Type for the file.

Routing Conditions

  • ip_address: filters requests based on their IP address.
  • host: filters requests based on the Host header.
  • referer: filters requests based on the Referer header.
  • campaign: filters requests based on whether their IP address is targeted by a Campaign.

def self.quit!(server,handler_name)
  # Use thins' hard #stop! if available, otherwise just #stop
  server.respond_to?(:stop!) ? server.stop! : server.stop

  print_info "Stopping Web Server on #{bind}:#{port}"
end

def self.run!(options={})
  set(options)

  handler      = detect_rack_handler
  handler_name = handler.name.gsub(/.*::/, '')

  runner = lambda { |handler,server|
    print_info "Starting Web Server on #{bind}:#{port}"
    print_debug "Using Web Server handler #{handler_name}"

    begin
      handler.run(server,Host: bind, Port: port) do |server|
        trap(:INT)  { quit!(server,handler_name) }
        trap(:TERM) { quit!(server,handler_name) }

        set :running, true
      end
    rescue Errno::EADDRINUSE => e
      print_error "Address is already in use: #{bind}:#{port}"
    end
  }

  if options[:background]
    Thread.new(handler,self,&runner)
  else
    runner.call(handler,self)
  end

  return self
end
