Overview

Represents an Operating System and pre-defines other common ones (OS.linux, OS.freebsd, OS.openbsd, OS.netbsd, OS.osx, OS.solaris, OS.windows and OS.unix).

def recent_ip_address
  relation = self.os_guesses.order('created_at DESC').first

  if relation
    return relation.ip_address
  end
end
