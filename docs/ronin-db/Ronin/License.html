Overview

Represents a software license and pre-defines many other common ones (License.bsd, License.cc_by, License.cc_by_sa, License.cc_by_nd, License.cc_by_nc, License.cc_by_nc_sa, License.cc_by_nc_nd, License.cc0, License.gpl2, License.gpl3, License.lgpl3 and License.mit).
