#erb(template) ⇒ String

Renders the inline ERB template in the scope of the object.

Examples:

@user = 'lolcats'

erb %{
USER: <%= @user %>
PASSWORD: <%= @user.reverse %>
}
# => "\nUSER: lolcats\nPASSWORD: staclol\n"

def erb_file(template_path)
  read_template(template_path) do |template|

  end
