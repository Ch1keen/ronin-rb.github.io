#pack(argument) ⇒ String

Packs the Float into a String.

Examples:

using Array#pack template:

0.42.pack('F')
# => =\n\xD7>"

using Ronin::Binary::Template types:

0x42.pack(:float_be)
# => ">\xD7\n="

Parameters:

  • argument (String, Symbol) —

    The Array#pack String code or Ronin::Binary::Template type.

Returns:

  • (String) —

    The packed float.

Raises:

  • (ArgumentError) —

    The given Symbol could not be found in Ronin::Binary::Template::FLOAT_TYPES.

def pack(argument)
  case argument
  when String
    [self].pack(argument)
  else
    unless Ronin::Binary::Template::FLOAT_TYPES.include?(argument)
      raise(ArgumentError,"unsupported integer type: #{argument}")
    end

    [self].pack(Ronin::Binary::Template::TYPES[argument])
  end
end
